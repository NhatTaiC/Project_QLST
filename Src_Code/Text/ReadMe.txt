Cài các ứng dụng cần thiết để chạy chương trình QLST:
+ Visual Studio 2022
+ SQL 2022
+ SSMS 19
+ Crystal Report SP32 for VS2022 (32 bit & 64 bit)

1. Vui lòng chạy file create_qlst_data.bat trong thư mục Data trước!
2. Sửa lại serverName khi chạy ứng dụng bị lỗi connection.
3. Tài Khoản: admin / Pass: 123456 / Chức Vụ: Admin

Note: Nếu chạy file create_qlst_data.bat bị lỗi, 
vui lòng Attach thủ công 2 file QLST.mdf và QLST_log.mdf hoặc QLST.bak 
trong thư mục Data vào SSMS 19
